# Next.js app with WordPress support starter

## Configuration

### Step 1. Set up your config

Copy the `config/default.json` to `config/local.json` and set `apiUrl` property to your WordPress GraphQL endpoint

### Step 2. Run Next.js in development mode

```bash
npm install
npm run dev

```

Your app should be up and running on [http://localhost:3000](http://localhost:3000)!
